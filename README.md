# gatsApi
GatsApi is a JavaScript library designed to enable developers to programmatically interact with the gats.io videogame via websockets. 

## Tutorial
To use GatsApi in your userscripts, you must do the following:
- Hook your player's websocket
- Intialize GatsApi by typing this: `const gatsApi = new GatsApi(websocket)`
- Now you can use commands such as gatsApi.game.sendChatMessage() or gatsApi.events.addEventListener()
